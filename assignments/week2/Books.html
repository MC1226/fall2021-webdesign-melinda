"JavaScript for Cats"

Summary: This web page is an introduction for new programmers. The author of this webpage, Max Ogden makes JavaScript very easy to understand and learn. I like the fact that he has moving commands that show the commands in gif form for each subject he is talking about such as The console, Strings, Values and variables, Using functions, Built in JS functions, Downloading new fuctions, Loops, Arrays, Objects, Callbacks which is very helpful.

He shows you how to get interactive in his webpage by having you open up a Console/Terminal straight from his webpage and can be done on any website and gives you commands to type and while you're entering the info his gif shows you how to do it the right way and what happens if you do it the wrong way. It's very interactive. He teaches you the basics and helps you understand it in simple terms, not complicated at all, he makes you want to learn and makes you want to keep reading because he's funny the way he uses cat references, along with cat images throughout his webpage and makes it enjoyable to read.

Towards the end the author puts a link to his github so if anyone wants to see any other topic covered they can open an issue on his github and he will cover that issue. I like the fact that at the end he lists books, ebooks and guides as resources including a link to NodeSchool that he helped create. NodeSchool is an open source project run by volunteers with two goals: to create high quality programming curriculum and to host community learning events. I liked the author's sense of humor, at one point in the webpage he put an Author's note that said: I @maxogden do happily accept random cat photo tweets. I really found the web page very useful and enjoyed it very much. I'm sure that anyone who reads it will find it useful as well. It is available to everyone and his goal is to teach everyone that beginning JavaScript is obtainable.
